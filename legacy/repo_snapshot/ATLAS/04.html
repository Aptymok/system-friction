Todo sistema llega a un borde.
No para detenerse,
sino para transformarse.
UMBRAL SISTÉMICO
El umbral marca el momento en que el sistema deja de escalar cuantitativamente y comienza a escalar cualitativamente.
dominioₜ₊₁ = transformación( dominioₜ )
[ pliego · expansión ]
Lo que era unidad,
ahora es campo.
EXPANSIÓN DE ESCALA
La escala no aumenta el número de elementos:
aumenta el rango posible de interacción.
escala' = f( coherencia_interna , tensión_residual )
La expansión vuelve hacia dentro.
Pero no vuelve igual.
RESONANCIA SISTÉMICA
La resonancia ocurre cuando los cambios del dominio ampliado comienzan a modificar las capas que lo originaron.
retroimpacto = f( dominioₙ₊₁ → Sₙ )
[ pliego · retropropagación ]
El afuera modifica al adentro,
y da origen a algo nuevo.
RETROPROPAGACIÓN
El dominio superior altera la estructura original.
Sₙ' = Sₙ + Δ( dominioₙ₊₁ )
Nada permanece igual.
Pero algo permanece.
PERSISTENCIA SISTÉMICA
La persistencia no es inercia: es identidad a través del cambio.
persistencia = estabilidad( patrón_interno )
Todo lo que el sistema fue,
converge en una nueva figura.
COHERENCIA TOTAL
La culminación es la estabilización total de todas las transformaciones previas: expansión, resonancia, persistencia y apertura.
coherencia_total = convergencia( IHG , NTI , R , LDI )
[ pliego · cierre del ciclo ]
Lo que abre el ciclo,
vuelve para cerrarlo,
pero en otra escala.
CIERRE DEL CICLO
El sistema regresa al punto inicial con estructura transformada: no repetición, sino integración.
S_final = f( S_inicial , Σ transformaciones )
