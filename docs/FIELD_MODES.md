# FIELD MODES

## SFI

Vista del sistema completo. Muestra los nodos de friccion, campo, latencia, resonancia, MIHM y capacidades.

## CT

Cognitive Twin general. Inicia desde andamio SFI y extrae patrones desde actividad, decisiones, contradicciones, omisiones y acciones.

Visible:

`El sistema detecta continuidad con patrones previos.`

Nunca visible:

`tu cognitive twin detecto...`

## NODE CT

Gemelo de nodo. Al seleccionar un nodo muestra:

- Patron dominante.
- Bloqueo.
- Accion minima.
- Dependencias.
- Consecuencia de no moverlo.
- Ruta sugerida.

## Barra inferior

La barra es el unico input estable. Cambia de modo por el nodo activo, no de posicion.
