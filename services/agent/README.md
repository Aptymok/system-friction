# services/agent

Servicio del Agente SFI: memoria, propuestas y autoconocimiento operacional.

Responsabilidad declarada:
- generar propuestas e inferencias trazables;
- consumir `FieldState`, `NodeState` y `Logs`;
- emitir eventos epistemicos con evidencia y confianza.

Limites:
- no finge conciencia;
- no declara verdad del campo;
- no ejecuta cambios sin autorizacion;
- CognitiveTwin permanece experimental hasta demostrar funcionalidad y trazabilidad.
