0 · Ontología del nodo
Un sistema no comienza a fallar
cuando algo se rompe.

Comienza a fallar cuando algo se observa,
se nombra,
pero nadie lo registra.
MUSICSCORE no es un DAW.
Es la primera instancia donde la fricción sistémica
de una organización musical se convierte en señal audible.

El sonido que produce no es diseñado.
Es emergente. Sale del estado real del sistema.

Si el sistema está en hegemonía extrema,
el kick baja de frecuencia y el patrón se fragmenta.
Si la coherencia institucional sube,
la melodía encuentra su motivo.

II · La función de costo global
J  =  w₁(IHG + 0.38)²  +  w₂(1 − NTI)²  +  w₃(1 − R)²  +  w₄·CFF²
w₁ = 1.0  ·  w₂ = 0.8  ·  w₃ = 0.6  ·  w₄ = 0.4
Objetivo homeostático:   IHG → −0.38  ·  J → 0
u(t)  =  Kp·e(t)  +  Ki·∫e(τ)dτ  +  Kd·de/dt
Kp = 4.2  ·  Ki = 0.35  ·  Kd = 1.3  ·  e(t) = IHG − (−0.38)
Meta-control automático cada 80 iteraciones
IRC  =  corr(time, J_history)  ∈  [0, 1]
IRC > 0.65  → sistema homeostático  ·  IRC < 0.35  → sistema desconectado
