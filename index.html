El problema real

La fricción no duele.
Sangra despacio.

Los sistemas no colapsan de golpe. Se vacían por fricción acumulada — la distancia entre lo que el sistema sabe y lo que ejecuta, entre lo que dice que importa y lo que sostiene con su comportamiento.

Esa distancia tiene nombre. Tiene variables. Y tiene un punto de intervención.

IHG
Índice de Homeostasis y Gobernanza
La distancia entre el estado declarado y la realidad observable. Cuanto mayor la brecha, mayor la carga entrópica acumulada.
NTI
Nodo de Trazabilidad Interna
¿La señal que el sistema tiene sobre sí mismo llega sin distorsión al punto de decisión? Si no llega, el sistema no puede gobernarse.
LDI
Latencia de Detección e Intervención
Tiempo entre la emergencia del problema y el primer intento de acción. Si LDI respuesta >> LDI detección: el problema no es de conocimiento. Es de ejecución.

Para quién

El sistema opera en
cualquier escala de fricción

Founder / Emprendedor
El que construye el sistema y vive dentro de él
Alta actividad, bajo efecto. Sabe lo que debería hacer. No lo ejecuta. El atractor está fuera del alcance de las herramientas disponibles.
"Hago mucho y no llego a ningún lado."
Directivo / Líder
El que toma decisiones con información distorsionada
NTI comprometido. La señal que el sistema tiene sobre sí mismo no llega sin distorsión al nivel de decisión. Los resultados no coinciden con las expectativas.
"Las cosas no pasan aunque yo lo pida."
Profesional / Operativo
El que conoce el problema y no puede moverlo
LDI crítico. La detección ya ocurrió. El problema es la respuesta. Hay una decisión que lleva tiempo sin tomarse y cuyo costo de postergación ya superó el costo de decidir.
"Sé qué está mal. No sé qué hacer primero."
