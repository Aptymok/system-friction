# System Friction Institute · Foundational Package

This ZIP contains the first editorial and operational package for the SFI Longitudinal Observation Framework.

## Contents

- `SFI-DT-001_Longitudinal_Observation_Framework.html` / `.pdf` — official institutional publication.
- `SFI-WB-001_Operator_Workbook.html` / `.pdf` — workbook for the frictionnaut/operator.
- `SFI-WB-002_Participant_Workbook.html` / `.pdf` — participant-facing workbook.
- `assets/sfi-editorial-system.css` — editorial system stylesheet.
- `assets/sfi-mark.svg` — vector mark.
- `sfi-atlas-schema.json` — prediction/atlas schema.
- `sfi_prediction_registry.py` — timestamped prediction registry.
- `index.html` — package landing page.

## Print

Open any HTML file in Chrome and print to PDF with background graphics enabled, or use the included PDFs.

## Core rule

Without a prediction registered before perturbation, the result is not predictive evidence. It is retrospective observation.
