# packages/campo-ob

CAMPO-OB: bitacoras, nodos, vinculos y regimen del campo.

Responsabilidad declarada:
- definir el modelo constitucional del campo;
- representar nodos, vinculos, logs y salud de fuentes;
- producir contratos consumibles por apps sin acceso directo a DB.

Limites:
- no contiene UI;
- no accede a Supabase/Postgres;
- no calcula verdad desde componentes visuales;
- FASE 2A solo declara tipos base.
