# packages/api-contracts

OpenAPI/Zod schemas y contratos de eventos.

Responsabilidad declarada:
- definir contratos entre apps, services y packages;
- versionar comandos, eventos y respuestas;
- impedir acceso directo de aplicativos a la base de datos.

Limites:
- FASE 2A no introduce Zod schemas ejecutables;
- no modifica APIs existentes;
- no genera OpenAPI todavia;
- solo declara tipos base.
